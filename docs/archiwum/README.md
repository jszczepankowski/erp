# Archiwum dokumentów wykonawczych

Ten katalog zawiera dokumenty pomocnicze/listy kontrolne użyte podczas realizacji wdrożenia v1.

Zasada:
- dokumenty kanoniczne (status globalny, decyzje końcowe, plan bazowy) pozostają w `docs/`,
- dokumenty wykonane/robocze (task-listy, checklisty krokowe) mogą być przenoszone do `docs/archiwum/` po formalnym zamknięciu etapu.
